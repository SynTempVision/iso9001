- scope of the certification
   - desigin, raw material, manufacturing, clean up / testing, assembly, verification testing, distribution
``` 
Customer Order
↓
Order Review (QA)
• specs complete?
• range, tolerance, quantity confirmed?
• unclear → stop & clarify
↓
Material Selection (QA)
• materials verified suitable
• visual check
↓
Manufacturing / Machining
↓
Assembly / Fabrication
↓
Calibration / Verification (QC)
• defined acceptance criteria
• pass / fail decision
• result recorded
↓
Labeling & Identification
• job / WO identification
• traceability as applicable
↓
Final Review & Packaging (QC)
• correct item
• correct customer
• documentation present
↓
Shipping

```
